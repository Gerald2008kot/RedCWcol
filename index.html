Para mantener RedCW genial para todos, necesitamos que aceptes nuestras leyes.

1. Respeto ante todo: No toleramos el ciberacoso, doxxing o amenazas. Somos una comunidad, no un campo de batalla.

2. Contenido Seguro: Prohibido subir material ilegal, explícito (NSFW) o violencia gráfica.

3. Moderación y Anonimato: Las funciones de anonimato son para libre expresión, no para atacar. Nos reservamos el derecho de banear IPs por mal uso.

4. Responsabilidad: Eres dueño de lo que publicas. RedCW no se hace responsable de las opiniones de los usuarios.
